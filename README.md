# LMS AI Solver — prod channel

This public repository contains only the installable userscript shell and its
version metadata. The application source remains in the private
`mobius_solver` repository.

## Install

[Install the prod channel](https://raw.githubusercontent.com/scrxpted7327/lms-solver/refs/heads/main/install.js)

Release branches:

- `main`: production
- `beta`: beta and release-candidate builds
- `dev`: development builds

The shell requires a GitHub personal access token with access to the private
source repository. It also requires the local backend documented there.

These files are generated by `scripts/release.py` in the private repository;
do not edit them directly.
